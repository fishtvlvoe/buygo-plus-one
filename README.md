# BuyGo+1

A complete WordPress product and order management system integrated with LINE.

## Features

- LINE product listing with image and text support
- Order management and tracking
- Shipment management
- Product search
- Inventory allocation
- Flexible settings system

## Requirements

- WordPress 6.5+
- PHP 7.4+

## Installation

1. Upload the plugin folder to `/wp-content/plugins/buygo-plus-one/`
2. Activate the plugin in the WordPress admin
3. Visit `yoursite.com/buygo-portal/dashboard` to get started

## Tech Stack

- Frontend: Vue 3, Tailwind CSS, React
- Backend: PHP 7.4+, WordPress REST API
