# AI Writer 文章模板

各平台的文章結構模板和格式規範。

---

## 部落格文章模板

**字數**：1000-2000 字
**特色**：SEO 優化、結構清晰、深度內容

### 結構模板

```markdown
# [吸引人的標題，包含關鍵字]

[開頭 Hook：150-200 字]
- 用問題、數據或故事吸引讀者
- 點出痛點或好奇點
- 預告文章價值

---

## [重點一標題]

[段落內容：200-300 字]
- 說明概念
- 提供例子
- 連結到讀者經驗

## [重點二標題]

[段落內容：200-300 字]
- 深入探討
- 實用建議
- 數據或案例支持

## [重點三標題]

[段落內容：200-300 字]
- 進階技巧
- 常見問題
- 專家觀點

---

## 結語

[總結 + CTA：100-150 字]
- 重申核心價值
- 行動呼籲
- 邀請互動（留言、分享）
```

### SEO 檢查清單

- [ ] 標題包含主要關鍵字
- [ ] H2 標題包含次要關鍵字
- [ ] 開頭 100 字內出現關鍵字
- [ ] 內文自然分布關鍵字
- [ ] 結尾有明確 CTA

## Facebook 貼文模板

**字數**：100-300 字
**特色**：情緒化開頭、互動問句、適量 emoji

### 結構模板

```markdown
[Hook：情緒化開頭，15-30 字]
用問題、驚嘆或共鳴開場

[主體：70-200 字]
- 分享觀點或故事
- 用換行增加可讀性
- 適當使用 emoji 點綴

[CTA：15-50 字]
- 問問題引發互動
- 或邀請點擊連結
- 或請求分享

[Hashtag：3-5 個，可選]
```

### 範例結構

```
你有沒有過這種經驗？☕

[描述共鳴場景]
[分享個人觀點]
[提供簡單建議]

💬 你們都怎麼處理這種情況？
留言告訴我！

#相關標籤 #主題標籤
```

### 注意事項

- 第一行要有 Hook，因為會被截斷
- 段落要短，每 2-3 行換行
- emoji 不要過多，3-5 個為佳
- 問句結尾提高互動率

## Instagram 貼文模板

**字數**：50-150 字
**特色**：短句、視覺思維、hashtag 策略

### 結構模板

```markdown
[Hook：第一句話最重要]

[主體：簡短有力]
- 用短句
- 每句換行
- 視覺節奏感

[CTA：簡單直接]

---
.
.
.
[Hashtag 區塊：15-30 個]
```

### 範例結構

```
這就是改變的開始 ✨

有時候
一個小決定
就能帶來大不同

你準備好了嗎？

👉 連結在自介

.
.
.
#標籤1 #標籤2 #標籤3
[更多 hashtag]
```

### Hashtag 策略

| 類型 | 數量 | 範例 |
|-----|-----|-----|
| 大眾標籤 | 3-5 | #台灣 #生活 |
| 主題標籤 | 5-10 | #咖啡控 #職場 |
| 品牌標籤 | 1-3 | #你的品牌名 |
| 社群標籤 | 3-5 | #IGers台灣 |

## LinkedIn 貼文模板

**字數**：200-500 字
**特色**：專業語氣、數據佐證、職場觀點

### 結構模板

```markdown
[Hook：專業但有溫度的開場]

[背景/問題描述]
- 點出職場痛點
- 或分享產業觀察

[核心觀點/解決方案]
- 條列式呈現
- 搭配數據或案例

[個人見解/呼籲]
- 分享你的看法
- 邀請專業討論

[CTA：職場互動]

#職場相關標籤 #產業標籤
```

### 範例結構

```
上週我做了一個決定，改變了整個團隊的工作方式。

[情境描述]

這讓我學到 3 件事：

1️⃣ [學習一]
2️⃣ [學習二]
3️⃣ [學習三]

最重要的是，[核心洞察]

你在職場上有過類似的經驗嗎？
歡迎在留言區分享你的看法。

#領導力 #團隊管理 #職場成長
```

### 注意事項

- 開頭 3 行決定會不會被展開
- 數字和條列增加可讀性
- 避免太過銷售導向
- 展現專業但要有人味

## Email 行銷模板

**字數**：300-600 字
**特色**：主旨行、個人化、明確 CTA

### 結構模板

```markdown
**主旨行**：[吸引開信的標題]
**預覽文字**：[補充主旨的 40-60 字]

---

[稱呼]

[開頭 Hook：1-2 句]
- 建立連結
- 點出價值

[主體內容]
- 說明重點（不超過 3 個）
- 用條列或短段落
- 保持聚焦

[CTA 區塊]
- 明確的行動按鈕
- 說明點擊後會發生什麼

[結尾]
- 簡短收尾
- 簽名

---

[P.S. 可選：補充資訊或額外誘因]
```

### 主旨行公式

| 類型 | 公式 | 範例 |
|-----|-----|-----|
| 好奇型 | 問題 + 懸念 | 你知道 90% 的人都搞錯了嗎？ |
| 利益型 | 數字 + 好處 | 3 個方法讓你省下 50% 時間 |
| 緊迫型 | 時間 + 行動 | 限時 24 小時：獨家優惠 |
| 故事型 | 人物 + 結果 | 她用這招，業績翻了 3 倍 |

### 範例結構

```
主旨：[Name]，這個方法幫了我很大的忙

---

Hi [Name]，

上週我遇到一個問題...
[簡短故事]

後來我發現一個方法：
• 重點一
• 重點二
• 重點三

如果你也想試試，
👉 [點這裡查看詳情]

有任何問題都可以直接回信問我。

Best,
[你的名字]

P.S. 這個優惠只到本週五喔！
```

## 模板選擇邏輯

```
使用者指定平台 → 直接使用對應模板
使用者未指定 → 詢問目標平台
使用者說「都要」→ 先產出一版，再改寫其他版本
```

## 跨平台改寫原則

從長到短改寫較容易：

```
部落格（1500字）
    ↓ 摘要重點
LinkedIn（400字）
    ↓ 精簡+情緒化
Facebook（200字）
    ↓ 極簡+視覺化
Instagram（100字）
```

從短到長需要擴充：

```
Instagram（100字）
    ↓ 加入更多細節和例子
Facebook（200字）
    ↓ 加入數據和深度分析
LinkedIn（400字）
    ↓ 加入完整論述和案例
部落格（1500字）
```
