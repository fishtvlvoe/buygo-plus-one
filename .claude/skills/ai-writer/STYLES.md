# AI Writer 語氣風格指南

定義各種語氣風格的特徵和使用場景。

---

## 語氣風格一覽

| 風格 | 適用場景 | 特徵 |
|-----|---------|-----|
| 專業 | B2B、LinkedIn、白皮書 | 數據導向、客觀、術語 |
| 親切 | 社群、Email、品牌溝通 | 對話感、同理心、生活化 |
| 活潑 | IG、年輕族群、促銷 | 短句、emoji、驚嘆 |
| 權威 | 教育內容、專家分享 | 自信、有條理、引用來源 |
| 溫暖 | 故事型、品牌形象 | 情感連結、個人經驗 |

---

## 專業風格

### 特徵
- 使用產業術語（但要解釋）
- 引用數據和研究
- 客觀陳述，避免主觀形容
- 結構嚴謹，邏輯清晰

### 用詞範例

| 避免 | 使用 |
|-----|-----|
| 超級厲害 | 顯著提升 |
| 很多人 | 根據調查，78% 的使用者 |
| 感覺很棒 | 帶來正面效益 |
| 應該可以 | 預期能夠 |

### 句型範例

```
❌ 這個方法超好用！
✅ 此方法經實測可提升 40% 效率。

❌ 大家都說很有效
✅ 根據 2025 年產業報告顯示，採用此策略的企業平均成長 25%。
```

## 親切風格

### 特徵
- 像朋友對話的語氣
- 使用「你」、「我們」
- 展現同理心
- 生活化的比喻

### 用詞範例

| 避免 | 使用 |
|-----|-----|
| 使用者應該 | 你可以試試 |
| 本公司提供 | 我們準備了 |
| 務必注意 | 記得留意一下 |
| 效益最大化 | 讓效果更好 |

### 句型範例

```
❌ 消費者應詳閱使用說明。
✅ 記得先看一下說明書，會更順手喔！

❌ 本服務提供多元解決方案。
✅ 不管你遇到什麼問題，我們都有辦法幫你。
```

---

## 活潑風格

### 特徵
- 短句、節奏快
- 適當使用 emoji
- 驚嘆句、問句
- 口語化表達

### 用詞範例

| 避免 | 使用 |
|-----|-----|
| 非常推薦 | 必須推！|
| 很好看 | 美炸！|
| 請參考 | 快來看！|
| 令人驚訝 | 太扯了吧！|

### 句型範例

```
❌ 這款產品的設計非常精美。
✅ 這設計也太美了吧！😍

❌ 我們的服務獲得許多好評。
✅ 大家都在問這是哪裡買的！🔥
```

### Emoji 使用指南

| 情境 | 推薦 emoji |
|-----|-----------|
| 重點強調 | ✨ 💡 ⭐ |
| 正面情緒 | 😊 🥰 💪 |
| 行動呼籲 | 👉 📲 🔗 |
| 警示提醒 | ⚠️ 📢 🔔 |
| 列表標記 | ✅ • 1️⃣ |

## 權威風格

### 特徵
- 自信但不傲慢
- 有條理的論述
- 引用可信來源
- 明確的觀點立場

### 用詞範例

| 避免 | 使用 |
|-----|-----|
| 我覺得 | 根據我的經驗 |
| 可能是 | 關鍵在於 |
| 大概這樣 | 具體做法如下 |
| 應該吧 | 建議採取 |

### 句型範例

```
❌ 我覺得這樣做可能會比較好。
✅ 根據我過去 10 年的經驗，這個策略是最有效的。

❌ 聽說這個方法不錯。
✅ 哈佛商業評論指出，此方法可提升團隊效率 35%。
```

---

## 溫暖風格

### 特徵
- 情感連結優先
- 分享個人故事
- 表達理解和支持
- 鼓勵性的語氣

### 用詞範例

| 避免 | 使用 |
|-----|-----|
| 你應該 | 當你準備好的時候 |
| 必須做到 | 慢慢來，沒關係 |
| 很簡單 | 一步一步來 |
| 不要擔心 | 我懂那種感覺 |

### 句型範例

```
❌ 不要害怕失敗，趕快開始。
✅ 我也曾經害怕失敗，但後來發現，每一步都是學習。

❌ 你必須克服這個困難。
✅ 這條路不容易，但你不是一個人。
```

## 風格選擇邏輯

### 根據平台選擇

| 平台 | 預設風格 | 備選 |
|-----|---------|-----|
| 部落格 | 專業/權威 | 親切 |
| Facebook | 親切 | 活潑、溫暖 |
| Instagram | 活潑 | 溫暖 |
| LinkedIn | 專業/權威 | 親切 |
| Email | 親切 | 專業 |

### 根據目的選擇

| 目的 | 推薦風格 |
|-----|---------|
| 導購 | 親切 + 活潑 |
| 品牌形象 | 溫暖 + 專業 |
| 教育內容 | 權威 + 親切 |
| 互動參與 | 活潑 + 親切 |

### 根據讀者選擇

| 讀者 | 推薦風格 |
|-----|---------|
| 年輕族群 | 活潑 |
| 專業人士 | 專業/權威 |
| 一般消費者 | 親切 |
| 高階主管 | 專業 + 溫暖 |

## 風格調整指令

當使用者要求調整語氣時：

| 使用者說 | 調整方向 |
|---------|---------|
| 更專業 | 減少口語、加入數據 |
| 更活潑 | 加入 emoji、短句 |
| 更親切 | 用「你」、加入對話感 |
| 更正式 | 移除口語、結構化 |
| 更有溫度 | 加入故事、情感連結 |

## 繁體中文寫作注意事項

### 標點符號
- 使用全形標點：，。！？、
- 引號使用「」而非 ""
- 書名號使用《》

### 常見用詞
- 「的」「得」「地」正確使用
- 避免過度使用「的」
- 「因為...所以...」的正確配對

### 避免簡體中文詞彙

| 避免（簡體用法） | 使用（繁體用法） |
|----------------|----------------|
| 視頻 | 影片 |
| 軟件 | 軟體 |
| 信息 | 資訊 |
| 網絡 | 網路 |
| 數據 | 資料/數據 |
| 用戶 | 使用者 |
