<feature>
  <name>事件分組摺疊式顯示</name>
  <files>
    includes/admin/tabs/developer-tab.php
  </files>
  <behavior>
    PHP 端（資料準備）：
    1. 移除現有的 event_type_filter GET 參數篩選邏輯
    2. 從 WebhookLogger 取得所有事件（limit 改大或分批取）
    3. 用 PHP 對 $webhook_logs 按 event_type 做 groupBy
    4. 每個分組計算：計數(count)、最早時間(earliest)、最近時間(latest)

    HTML 端（渲染）：
    1. 移除篩選下拉選單 (bgo-dev-filters)
    2. 改為分組區塊，每個分組有：
       - 可點擊的標題列：[badge] event_type | N 筆 | 時間範圍
       - 摺疊/展開箭頭圖示
    3. error / permission_denied 預設展開 (open 屬性)
    4. 展開後顯示事件明細表格（前 20 筆）
    5. 如果該分組超過 20 筆，顯示分頁導航

    JS 端（互動）：
    1. 點擊分組標題 → 切換展開/收合
    2. 分頁按鈕 → AJAX 請求載入該分組的下一頁事件
       - 新增 AJAX handler: buygo_dev_event_page
       - 接收 event_type + page 參數
       - 回傳 HTML 片段（事件列表 + 分頁按鈕）

    AJAX Handler（新增）：
    - action: buygo_dev_event_page
    - 參數：event_type (string), page (int), per_page (int, 預設 20)
    - 回傳：事件列表 HTML + 分頁資訊
    - 權限：manage_options
    - Nonce: buygo_dev_nonce（複用現有 nonce）

    注意事項：
    - Section 2 (Data Cleanup) 和 Section 3 (SQL Console) 完全不動
    - CSS 新增的類別用 .bgo-event-group- 前綴
    - 保持 developer-tab.php 在同一檔案（維持現有慣例）
    - 分頁用 AJAX 而非全頁刷新（SPA 體驗）
  </behavior>
