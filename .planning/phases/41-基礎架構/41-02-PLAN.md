# Plan 41-02: Tab 導航 CSS

**Phase:** 41 - 基礎架構
**Goal:** 建立 `.bgo-` 前綴的 Tab 導航 CSS，結構對齊 LineHub 但使用 BGO 品牌藍

---

## Context

LineHub 的 Tab CSS（`line-hub/assets/css/admin-tabs.css`）使用 `.line-hub-` 前綴：
- `.line-hub-tabs` 容器（border-bottom + 白色背景）
- `.line-hub-tab a` 連結（padding: 12px 20px, border-bottom: 2px solid transparent）
- `.line-hub-tab.active a` 啟用狀態（品牌綠 `#06C755`）
- `.line-hub-card` 卡片

BGO 需要相同結構，品牌色改為 `#3b82f6`（品牌藍）。

## Tasks

### Task 1: 建立 admin/css/admin-tabs.css

**File:** `admin/css/admin-tabs.css`（新建）

```css
/**
 * BGO Admin Tabs Styles
 *
 * Tab 導航和佈局樣式 — 結構對齊 LineHub，品牌色為 BGO 藍
 *
 * @package BuyGoPlus
 * @since 2.0.0
 */

/* Tab 導航容器 */
.bgo-tabs {
    margin: 20px 0 0;
    border-bottom: 1px solid #ccc;
    background: #fff;
}

.bgo-tabs-wrapper {
    display: flex;
    align-items: center;
    padding: 0;
    margin: 0;
    list-style: none;
}

/* Tab 項目 */
.bgo-tab {
    margin: 0;
    padding: 0;
}

.bgo-tab a {
    display: block;
    padding: 12px 20px;
    text-decoration: none;
    color: #555;
    border-bottom: 2px solid transparent;
    transition: all 0.2s ease;
    font-weight: 500;
}

.bgo-tab a:hover {
    color: #3b82f6;
    background: #f0f7ff;
}

.bgo-tab.active a {
    color: #3b82f6;
    border-bottom-color: #3b82f6;
    background: transparent;
}

/* Tab 內容區域 */
.bgo-tab-content {
    margin-top: 20px;
}

/* 卡片樣式（對齊 LineHub .line-hub-card） */
.bgo-card {
    background: #fff;
    border: 1px solid #ccd0d4;
    box-shadow: 0 1px 1px rgba(0,0,0,.04);
    margin: 20px 0;
    padding: 20px;
    max-width: 1000px;
}

.bgo-card h2 {
    margin: 0 0 15px 0;
    padding-bottom: 10px;
    border-bottom: 1px solid #eee;
}

/* 狀態指示器（對齊 LineHub .line-hub-status-indicator） */
.bgo-status-indicator {
    display: inline-flex;
    align-items: center;
    gap: 6px;
    padding: 4px 12px;
    border-radius: 3px;
    font-size: 13px;
    font-weight: 500;
}

.bgo-status-indicator.success {
    background: #d4edda;
    color: #155724;
}

.bgo-status-indicator.error {
    background: #f8d7da;
    color: #721c24;
}

.bgo-status-indicator.warning {
    background: #fff3cd;
    color: #856404;
}

.bgo-status-indicator.info {
    background: #d1ecf1;
    color: #0c5460;
}

/* 響應式設計 */
@media screen and (max-width: 782px) {
    .bgo-tabs-wrapper {
        flex-wrap: wrap;
    }

    .bgo-tab a {
        padding: 10px 15px;
        font-size: 14px;
    }

    .bgo-card {
        padding: 15px;
    }
}
```
