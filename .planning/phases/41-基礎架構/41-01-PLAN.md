# Plan 41-01: 選單重構

**Phase:** 41 - 基礎架構
**Goal:** 將 BGO 後台從 2 個子選單合併為單一「BGO」選單頁面

---

## Context

目前 `class-settings-page.php:36-68` 註冊了：
- 主選單 `BuyGo+1` (slug: `buygo-plus-one`) → 渲染 LINE 模板
- 子選單 `LINE 模板` (slug: `buygo-plus-one`) → 與主選單同 slug
- 子選單 `設定` (slug: `buygo-settings`) → 渲染設定頁（6 Tab）

v2.0 需要合併為一個頁面，選單名稱從「BuyGo+1」改為「BGO」。

## Tasks

- WordPress 會自動建立一個與主選單同名的子選單項，保持不變即可

### Task 2: 建立統一的 render_page() 方法

**File:** `includes/admin/class-settings-page.php`

新增 `render_page()` 方法，取代 `render_settings_page()` 和 `render_templates_page()`：

```php
/**
 * 渲染 BGO 後台統一頁面
 */
public function render_page(): void
{
    // 處理表單提交（模板和設定共用）
    if (isset($_POST['submit']) && wp_verify_nonce($_POST['_wpnonce'], 'buygo_settings')) {
        $this->handle_form_submit();
    }
    if (isset($_POST['submit_templates']) && isset($_POST['templates']) && wp_verify_nonce($_POST['_wpnonce'], 'buygo_settings')) {
        $this->handle_templates_submit();
    }

    // 新 6-Tab 結構（預設 Tab 為角色權限）
    $current_tab = $_GET['tab'] ?? 'roles';
    $tabs = [
        'roles'     => '角色權限',
        'templates' => 'LINE 模板',
        'checkout'  => '結帳設定',
        'data'      => '資料管理',
        'features'  => '功能管理',
        'developer' => '開發者',
    ];

    ?>
    <div class="wrap">
        <h1>BGO</h1>
        <?php settings_errors('buygo_settings'); ?>

        <!-- Tab 導航（.bgo-tabs 樣式，對齊 LineHub） -->
        <div class="bgo-tabs">
            <ul class="bgo-tabs-wrapper">
                <?php foreach ($tabs as $tab_key => $tab_label): ?>
                    <li class="bgo-tab <?php echo $current_tab === $tab_key ? 'active' : ''; ?>">
                        <a href="?page=buygo-plus-one&tab=<?php echo esc_attr($tab_key); ?>">
                            <?php echo esc_html($tab_label); ?>
                        </a>
                    </li>
                <?php endforeach; ?>
            </ul>
        </div>

        <!-- Tab 內容 -->
        <div class="bgo-tab-content">
            <?php
            switch ($current_tab) {
                case 'roles':
                    $this->render_roles_tab();
                    break;
                case 'templates':
                    $this->render_templates_tab();
                    break;
                case 'checkout':
                    $this->render_checkout_tab();
                    break;
                case 'data':
                    // Phase 43 實作，目前顯示佔位
                    echo '<div class="bgo-card"><p>資料管理功能將在後續版本中實作。</p></div>';
                    break;
                case 'features':
                    // Phase 44 實作，目前顯示佔位
                    echo '<div class="bgo-card"><p>功能管理將在後續版本中實作。</p></div>';
                    break;
                case 'developer':
                    // Phase 45 實作，目前載入舊 Tab
                    $this->render_workflow_tab();
                    break;
            }
            ?>
        </div>
    </div>
    <?php
}
```
