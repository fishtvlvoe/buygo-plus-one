### Task 5: buygo_helper_can() 輔助函式

**File:** `includes/services/class-settings-service.php`

```php
/**
 * 檢查小幫手是否有特定權限
 * 賣家和 WP Admin 永遠回傳 true
 */
public static function helper_can(int $user_id, string $permission): bool
{
    $user = get_userdata($user_id);
    if (!$user) return false;

    // 賣家和 WP Admin 永遠有權限
    if (in_array('administrator', $user->roles) || in_array('buygo_admin', $user->roles)) {
        return true;
    }

    // 小幫手：檢查細粒度權限
    if (in_array('buygo_helper', $user->roles)) {
        $permissions = get_user_meta($user_id, 'buygo_helper_permissions', true);
        if (empty($permissions) || !is_array($permissions)) {
            return true; // 預設全部開啟（向後相容）
        }
        return !empty($permissions[$permission]);
    }

    return false;
}
```
