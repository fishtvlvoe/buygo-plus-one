**1. DashboardService 新增 calculateProfitStats()：**

```php
public static function calculateProfitStats(): array {
    // 1. 查詢有成本價的商品：JOIN fct_order_items + buygo_product_meta
    // 2. 計算每個商品的利潤：(item_subtotal - quantity × cost_price)
    // 3. 只算已完成訂單（order_status = 'completed'）
    // 4. 回傳：
    //    - total_profit（總利潤，分為單位）
    //    - total_revenue_with_cost（有成本資料的營收）
    //    - avg_profit_margin（平均利潤率 %）
    //    - top_products（Top 5 利潤商品：name, profit, margin）
    //    - by_currency（按幣別分組的利潤）
}
```

SQL 查詢邏輯：
```sql
SELECT
    oi.product_id,
    p.title as product_name,
    o.currency,
    SUM(oi.quantity) as total_quantity,
    SUM(oi.subtotal) as total_revenue,
    pm.meta_value as cost_price,
    SUM(oi.subtotal) - SUM(oi.quantity * CAST(pm.meta_value AS DECIMAL(10,2)) * 100) as profit
FROM {prefix}fct_order_items oi
JOIN {prefix}fct_orders o ON oi.order_id = o.id
JOIN {prefix}buygo_product_meta pm ON oi.product_id = pm.product_id AND pm.meta_key = 'cost_price'
LEFT JOIN {prefix}fct_products p ON oi.product_id = p.id
WHERE o.order_status = 'completed'
  AND o.mode = 'live'
  AND pm.meta_value != '' AND pm.meta_value != '0'
GROUP BY oi.product_id, o.currency
ORDER BY profit DESC
```

注意：cost_price 在 buygo_product_meta 中以「元」儲存（非分），fct_order_items.subtotal 以「分」儲存，需要轉換。

在 loadAllData() 的 Promise.all 中加入 loadProfit()：
```javascript
loadProfit() {
    return fetch('/wp-json/buygo-plus-one/v1/dashboard/profit', {
        credentials: 'include',
        headers: { 'X-WP-Nonce': this.wpNonce }
    })
    .then(res => res.json())
    .then(result => {
        if (result.success) {
            this.profitData = result.data;
            if (window.BuyGoCache) BuyGoCache.set('dashboard_profit', result.data);
        }
    });
}
```
