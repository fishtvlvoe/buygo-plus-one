- [ ] 打開 Browser DevTools (F12)
- [ ] 進入 Console 標籤

- [ ] 記錄任何錯誤：


#### 4. API 連線檢查
在 DevTools → Network 標籤，檢查 API 請求：

- [ ] `/wp-json/` 開頭的請求

記錄任何錯誤請求：

```

訊息：
```

#### 5. 功能基本檢查
