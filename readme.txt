BuyGo 獨立賣場後台系統

== Description ==

BuyGo+1 是完全獨立的 WordPress 外掛，提供 BuyGo 賣場後台管理功能。

== Installation ==

1. 上傳 `buygo-plus-one` 資料夾到 `/wp-content/plugins/` 目錄
2. 在 WordPress 後台的「外掛」選單中啟用 BuyGo+1
3. 訪問 `yoursite.com/buygo-portal/dashboard` 開始使用
