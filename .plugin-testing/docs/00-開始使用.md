✗ 擔心出錯
  每步都有備份，隨時可回滾

✗ 在多個地方編輯
  一個根目錄，一個框架，簡單清晰

## 🔒 安全機制

### 備份很簡單

```
AI會自動:
1. 修改前詢問「要備份嗎？」
2. git add .
3. git commit -m "備份信息"
4. 執行修改
5. 驗證結果

如果出問題:
git reset --hard HEAD~1
一個命令，回到之前的狀態
```

### 查看歷史

```bash
# 看所有備份點
git log --oneline

# 看詳細修改
git show abc1234
```
